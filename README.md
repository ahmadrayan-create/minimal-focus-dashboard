# 🌅 ZenFocus: Dynamic Productivity Dashboard

ZenFocus is a minimal, browser-based landing page designed to keep you centered and productive throughout the day. It dynamically changes its theme based on the time of day and remembers your name and goals even after you refresh the browser.

[🚀 View Live Demo](https://ahmadrayan-create.github.io/minimal-focus-dashboard/)

## ✨ Key Features
- **Time-Aware Greeting:** Automatically shifts between Morning, Afternoon, and Evening themes.
- **Persistent Data:** Uses `localStorage` to save your name and daily focus so your data stays put.
- **Motivational Engine:** Displays a fresh inspirational quote on every load.
- **Interactive UI:** Content-editable fields allow for seamless, click-to-type interaction.
- **Fully Responsive:** Beautifully optimized for desktops, tablets, and mobile devices.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Flexbox layout, glassmorphism effects, and media queries.
- **JavaScript (ES6+):** - Date & Time API for the live clock.
  - Local Storage for data persistence.
  - Dynamic DOM manipulation for theme switching.

## 📂 Project Structure
- `index.html` - Core structure and Google Fonts integration.
- `style.css` - Responsive styling and background transitions.
- `script.js` - Logic for time, storage, and dynamic styling.
- `/images` - Optimized background assets for different times of day.

## 🚀 Quick Start
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/repo-name.git`
2. Open `index.html` in your browser.
3. Click on "[Enter Name]" to personalize your dashboard!
